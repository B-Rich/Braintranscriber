#Braintranscriber
Brainfuck and Ook! interpterer and translator.
##Usage
```
bt [-options] [Brainfuck/Ook code]
  -b	interpret Brainfuck (default)
  -o	interpret Ook!
  -i	run interpreted code (default)
  -t	translate Brainfuck to Ook and vice versa
```
